# 水印watermark

#### 介绍
给各种文件添加水印。
add water mark to images ,pdf, powerpoint, word,excel
这是一个java实现对各种文件添加水印的代码示例，工具可以直接拿来用。
也可以复制代码到自己项目中修改后使用。
